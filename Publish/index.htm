HID Demo

Description

HID Demo is a Windows Form based ClickOnce application allowing you to test and inspect HID devices on Microsoft Windows.

Installation

Download and run the setup.

Source Code

It is part of an open source C# library project called SharpLibHid and available on CodePlex.

Binaries

This library is available as a NuGet package for use in your own projects.

Adopters

MediaPortal is using this library to handle remotes and gamepads input.
